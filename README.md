# Customer Churn Analysis using Excel & Power BI

## Objective
The objective of this project is to analyze customer data and identify factors that contribute to customer churn. The project helps businesses understand customer behavior and improve retention strategies.

## Tools Used
- Microsoft Excel
- Power BI

## Dataset
The dataset contains customer information such as:
- Customer ID
- Gender
- Age
- Tenure
- Contract Type
- Monthly Charges
- Total Charges
- Churn Status

## Project Workflow
1. Data Cleaning in Excel
2. Data Preparation
3. Data Import into Power BI
4. Dashboard Creation
5. Business Insights Generation

## Dashboard Features
- Total Customers
- Churn Rate Analysis
- Customer Segmentation
- Contract Type Analysis
- Monthly Charges Analysis
- Customer Retention Insights

## Key Findings
- Identified customers with a higher risk of churn.
- Analyzed the impact of contract types on customer retention.
- Compared churn patterns across different customer groups.
- Generated interactive visual reports for decision-making.
